# Teslo API

1. Clonar proyecto
```
yarn install
```
2. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
3. Cambiar las variables de entorno
4. Levantar la Base de Datos
```
docker-compose up -d
```
5. Levantar la API
```
yarn start:dev
```
6. Ejecutar SEED
```
http://localhost:3000/api/seed
```
